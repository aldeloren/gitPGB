Full Site

The UF Mobile Web provides essential UF information and services anytime, anywhere on your mobile device, with an interface optimized for on-the-go access.

This site automatically detects the device you're using and delivers an experience optimized for it. You're using the version optimized for the iPhone, iPod Touch, and Android devices.

The UF Mobile Web is provided by UF Information Technology, in cooperation with other departments, as a free service* to the UF community.
